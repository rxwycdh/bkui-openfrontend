bkui-openfrontend
===

基于蓝鲸bkui vue2.x模版修改而成的帐号密码登录版框架，与蓝鲸不耦合，可自由对接任意系统
## 使用

``` bash
# 本地运行蓝鲸前端开发脚手架的文档工程
npm run doc

# 构建蓝鲸前端开发脚手架的文档工程
npm run build:doc

# 本地开发
npm run dev

# 创建 `Dll bundle`
npm run dll

# 打包构建
npm run build

# 打包构建分析
npm run build:analyzer
```
